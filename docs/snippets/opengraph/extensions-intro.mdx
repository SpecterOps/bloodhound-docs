OpenGraph extends BloodHound beyond Microsoft-centric Attack Paths by letting you ingest data from other identity providers, developer platforms, device management systems, and custom data sources. 

All OpenGraph extensions start with the same JSON data payload structure for nodes, edges, and metadata. <Tooltip tip="A file that defines the structure of an OpenGraph extension, including node types, edge types, traversability behavior, and visual configurations.">Extension definition schemas</Tooltip> build on that foundation by defining how BloodHound should interpret and analyze that data.
