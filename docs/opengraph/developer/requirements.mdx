# Version Requirements

Full OpenGraph support requires a PostgreSQL graph backend and one of the following editions:
- BloodHound Enterprise (uses PostgreSQL)
- BloodHound CE v8.0.0+ with PostgreSQL
    - To switch backends in CE, set `"graph_driver": "pg"` in `bloodhound.config.json` and re-ingest.

_Note: While many OpenGraph features may work on a Neo4j graph database, there are functional and performance limitations (see the [OpenGraph FAQ](/opengraph/faq#why-is-it-taking-so-long-to-ingest-opengraph-data)). For full support, migrate to a PostgreSQL graph database backend.
