## Edge Schema

- Source: [Okta_User](../nodes/Okta_User), [Okta_Group](../nodes/Okta_Group), [Okta_Application](../nodes/Okta_Application)
- Destination: [Okta_User](../nodes/Okta_User), [Okta_Group](../nodes/Okta_Group), [Okta_Device](../nodes/Okta_Device)
- Traversable: ✅

## General Information

The traversable `Okta_OrgAdmin` edges represent Organization Administrator role assignments.
Organization Administrators can manage most organizational settings except for administrative role assignments and some security settings.

```mermaid
graph LR
    u1("Okta_User john\@contoso.com")
    u2("Okta_User alice\@contoso.com")
    g1("Okta_Group IT")
    d1("Okta_Device John's MacBook")
    u1 -- Okta_OrgAdmin --> u2
    u1 -- Okta_OrgAdmin --> g1
    u1 -- Okta_OrgAdmin --> d1
```
