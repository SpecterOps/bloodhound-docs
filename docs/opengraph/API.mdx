# API Endpoints

|  |   |   |
| --- | --- | --- |
| GET | /api/v2/customnode | Get all custom node types and their configurations. |
| GET | /api/v2/customnode/{node_type} | Get configuration for a specific node type. |
| POST | /api/v2/customnode | Create a new custom node type with configuration. |
| PUT | /api/v2/customnode/{node_type} | Update the configuration of an existing node type |
| DELETE| /api/v2/customnode/{node_type} | |
| POST | /api/v2/clear-database | Clears all elements of the database including Base Node Types and Custom Node Types |
