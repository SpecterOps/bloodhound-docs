## Global Shortcuts
These shortcuts work in both BloodHound Enterprise and Community Edition

| Page              | Command | Action |
| ----------------- | ------- | ------ |
| Global | Option/Alt + H | View keyboard shortcuts dialog |
| Global | Option/Alt + M | Toggle Dark Mode |
| Explore Page | Option/Alt + S | Jump to Node Search |
| Explore Page | Option/Alt + P | Jump to Pathfinding |
| Explore Page | Option/Alt + C | Focus Cypher Query Editor |
| Explore Page | Option/Alt + Shift + S | Save Current Query |
| Explore Page | Option/Alt + R | Run Current Cypher Query |
| Explore Page | Option/Alt + / | Search Current Nodes |
| Explore Page | Option/Alt + T | Toggle Table View |
| Explore Page | Option/Alt + I | Toggle Node Info Panel |
| Explore Page | Option/Alt + Shift + G | Reset Graph View |


## Enterprise Only Shortcuts

| Page              | Command | Action |
| ----------------- | ------- | ------ |
| Attack Paths Page | Option/Alt + Shift + R | Reset to Default View |
| Attack Paths Page | Option/Alt + . | Jump to Next Finding |
| Attack Paths Page | Option/Alt + , | Jump to Previous Finding |
| Attack Paths Page | Option/Alt + E | Jump to Environment Selector |
| Posture Page | Option/Alt + E | Jump to Environment Selector |
| Posture Page | Option/Alt + Z | Jump to Zone Selector |
| Posture Page | Option/Alt + F | Filter Table Data |
