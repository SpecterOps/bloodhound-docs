  ## Explore Workflow and Usability Improvements

  Work through **Explore** results with clearer controls and less unnecessary visual clutter.

  This release minimizes the search, pathfinding, and Cypher panel and closes the entity panel when a Cypher query returns multiple nodes, reduces repeated category labels in **Saved Queries**, standardizes close and minimize control placement, and makes node type information more visible in the entity panel for faster triage and Cypher authoring.

  ## Ingest and Post-Processing Performance Improvements

  Process large datasets with faster traversal paths and less post-processing churn.

  This release optimizes graph index coverage for common traversal paths and migrates key post-processing workflows to delta-based apply behavior, reducing unnecessary delete-and-recreate cycles for Owns/WriteOwner and Azure App Role calculations.
</Update>

<Update label="BloodHound" tags={["Fixed Issues"]}>
  ## API, Auth, and Audit

  - {/*BED-8094*/} Resolved an issue where filtering in **API Explorer** only matched endpoint tags and did not reliably include matching API paths.
  - {/*BED-7931*/} Resolved an issue where **API Explorer** showed an outdated request example for `POST /api/v2/clear-database` that did not include newer OpenGraph delete options.
  - {/*BED-7761*/} Resolved an issue where users created through SSO and OIDC sign-in did not consistently receive the correct ETAC environment access based on their role claims.
  - {/*BED-7516*/} Resolved an issue where deleting a client through `DELETE /api/v2/clients/{id}` did not generate expected audit log entries.

  ## Ingest and Graph Processing

  - {/*BED-8058*/} Resolved an issue where the Neo4j `FetchKinds` path did not return edge kinds, which could cause incomplete kind metadata during graph operations.
  - {/*BED-8034*/} Resolved an issue where cross-product processing could create unnecessary edges when component sets shared nested members.
  - {/*BED-8028*/} Resolved an issue where edge deletion by kind could fail against Neo4j backends.
  - {/*BED-7781*/} Resolved an issue where a single unresolved `property_match` in an ingest payload could cause other valid payload entries to fail in the same submission.
  - {/*BED-7700*/} Resolved an issue where extension upload upserts used a delete-and-recreate pattern instead of true upsert behavior, causing unnecessary ID sequence churn.
  - {/*BED-7483*/} Resolved an issue where ingest operations could fail with repeated `Error reading ingest file` failures tied to conflicting upsert behavior.
  - {/*BED-7463*/} Resolved an issue where concurrent ingest and changelog flush activity could deadlock and interrupt processing.

  ## OpenGraph Data Correctness

  - {/*BED-7932*/} Resolved an issue where OpenGraph findings did not display severity color because severity values were not returned correctly.
  - {/*BED-7768*/} Resolved an issue where selecting **All X** options for OpenGraph environment filters did not correctly include all environments for the selected schema.
  - {/*BED-7762*/} Resolved an issue where object IDs could appear with an appended timestamp in the entity panel after specific OpenGraph node upsert sequences.

  ## Analysis and Azure Data Accuracy

  - {/*BED-7936*/} Resolved an issue where PreAgg histogram processing did not correctly accumulate smaller buckets into the expected cumulative histogram output.
  - {/*BED-6773*/} Resolved an issue where certain Azure resources could be misclassified as Active Directory group types based on SID-suffix matching during post-processing.
  - {/*BED-4754*/} Resolved an issue where `AZAddOwner` relationships were not created as expected in Azure analysis results.

  ## Explore and Cypher

  - {/*BED-7860*/} Resolved an issue where pressing the space bar in the Cypher editor on the **Explore** page could fail to insert spaces as expected.
  - {/*BED-7939*/} Resolved an issue where long edge-type content in pathfinding filter dialogs could overflow and reduce readability.
  - {/*BED-7586*/} Resolved an issue where search help text contrast in **Explore** was too low in dark mode.
  - {/*BED-4761*/} Resolved an issue where selected nodes in **Explore** could not be unselected to return the graph to a clear state.
  - {/*BED-4495*/} Resolved an issue where arrowheads in **Explore** could become visually misaligned when multiple edges connected the same nodes.

  ## Navigation and Table Layout

  - {/*BED-7898*/} Resolved an issue where multiple table views had column header alignment inconsistencies.
  - {/*BED-7904*/} Resolved an issue where the navigation logo could shrink excessively as browser height decreased.
  - {/*BED-7881*/} Resolved an issue where the navigation area could become non-scrollable at reduced window heights.

  ## Data Collection Connectivity

  - {/*BED-7980*/} Resolved an issue where RC collectors running in Docker could fail to connect due to unsupported user-agent parsing behavior.
</Update>
