## Announcements

### BloodHound Enterprise Splunk App (v1.1.0)

We are excited to announce the v1.1.0 release of the [BloodHound Enterprise Splunk App](https://splunkbase.splunk.com/app/7818). The app allows you to visualize and analyze BloodHound data directly within Splunk, providing enhanced capabilities for security analysis and threat hunting. Notable features in this release include:

- New **Posture History** dashboard showing key security metrics and trends over time
- Support for proxy configurations
