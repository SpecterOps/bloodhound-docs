<Note>OpenHound is currently in beta and is under active development. Features, workflows and supported collectors may change as the framework evolves.</Note>


OpenHound is a standardized framework for running and building OpenGraph collectors and converters. Built
in Python and powered by the [DLT](https://dlthub.com/docs/intro) (Data Load Tool) library, OpenHound provides a consistent
workflow for collecting, processing, and converting data from any source into BloodHound-compatible graphs. OpenHound is
designed to be used as a containerized service and as a standalone CLI application.
