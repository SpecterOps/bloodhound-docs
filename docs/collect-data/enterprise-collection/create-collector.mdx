## Purpose

This guide explains how to create a BloodHound Enterprise collector client. BloodHound Enterprise supports two types of collector clients:

- **SharpHound Enterprise** - Collects data from Active Directory environments
- **AzureHound Enterprise** - Collects data from Entra ID environments

It should be used by Administrators when deploying SharpHound Enterprise or AzureHound Enterprise.

## Prerequisites

- Having deployed a BloodHound Enterprise Tenant
- Logged in as a user assigned a [role](/manage-bloodhound/auth/users-and-roles) authorized to create a collector client

## Process

The process to create a BloodHound Enterprise collector client consists of the following steps:

<Steps>
  <Step title="Open the Manage Clients page">
  In the left menu, click **Administration** > **Manage Clients**.
  </Step>

  <Step title="Create a client">
    1. On the right side of the page, click **Create Client**.

    1. Select a client from the drop-down.

        Options include:

        - **SharpHound Enterprise Client** for Active Directory data collection
        - **AzureHound Enterprise Client** for Entra ID data collection

          <Tabs>
            <Tab title="SharpHound Enterprise Client">
              Enter details for the SharpHound Enterprise Client and click **Create**.

              | Field | Required | Description |
              |-------|----------|-------------|
              | **Client Name** | Required | A descriptive name for the collector client |
              | **Collection Schedule** | Optional | |
              | **Advanced Options** | Optional | |

                <Frame>
                  <img
                    src="/images/data_collectors/create-sharphound.png"
                    alt="A view of the Create Client screen for SharpHound Enterprise"
                  />
                </Frame>
              </Tab>

            <Tab title="AzureHound Enterprise Client">
              Enter details for the AzureHound Enterprise Client and click **Create**.

              | Field | Required | Description |
              |-------|----------|-------------|
              | **Client Name** | Required | A descriptive name for the collector client |
              | **Collection Schedule** | Optional | |

                <Frame>
                  <img
                    src="/images/data_collectors/create-azurehound.png"
                    alt="A view of the Create Client screen for AzureHound Enterprise"
                  />
                </Frame>
            </Tab>
          </Tabs>
  </Step>
  
  <Step title="Save the client token">
  The pop-up window _Client Token Info_ will appear; follow the instructions in it - save the key before clicking **CLOSE**.

    <Tabs>
      <Tab title="SharpHound Enterprise Client">
        Copy the token information for the SharpHound Enterprise Client.

        <Note>The token information is required to [configure](/install-data-collector/install-sharphound/local-configuration) the collector client application.</Note>

          <Frame>
            <img
              src="/images/data_collectors/sharphound-client-token-info.png"
              alt="A view of the client token info screen for SharpHound Enterprise"
            />
          </Frame>
        </Tab>

        <Tab title="AzureHound Enterprise Client">
          Copy the token information for the AzureHound Enterprise Client.

          <Note>The token information is required to [configure](/install-data-collector/install-azurehound/create-configuration) the collector client application.</Note>

            <Frame>
              <img
                src="/images/data_collectors/azurehound-client-token-info.png"
                alt="A view of the client token info screen for AzureHound Enterprise"
              />
            </Frame>
        </Tab>
      </Tabs>
  </Step>
</Steps>

## Outcome

The collector client will appear in the clients table with a **Status** of **Unconfigured**.

<Frame>
  <img
    src="/images/data_collectors/create-collector-outcome.png"
    alt=""
  />
</Frame>

You can now use the token information to configure the collector client application. See [Install SharpHound Enterprise](/install-data-collector/install-sharphound) or [Install AzureHound Enterprise](/install-data-collector/install-azurehound) for more information.
